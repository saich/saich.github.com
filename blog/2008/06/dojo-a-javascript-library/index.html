Dojo: A Javascript Library

7 June, 2008

Been trying to look into Dojo Toolkit for a few days… The features are good and I wondered why it wasn’t the No.1 ??? But, as days passed, I found a few worse things with that toolkit……

  • It was heavy-weight toolkit.
  • It loads all the JS files (summing to 700KB) even though not at all necessary…
  • Requires a high efficient system .. An ordinary system cannot work well Dojo …
  • It takes up all the RAM of the system, hanging the system for a while….
So, I thought its better to shift to some other toolkit —– May be JQUERY is better ……
Let me try that also….
